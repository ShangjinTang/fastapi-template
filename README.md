# fastapi-template

## Prerequisites

Install `uv`.

- macOS and Linux:

    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```

- Windows:

    ```powershell
    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
    ```

## Steps to create this template

1. Create a `.python-version` file.

    ```bash
    echo "3.12" > .python-version
    ```
